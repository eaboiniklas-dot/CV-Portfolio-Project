Arden Rassloff CV Portfolio Website

 Project Overview

This project is an individual front-end development assessment portfolio website created using HTML, CSS, JavaScript, GitHub, and Bash scripting.

The website presents:
- A homepage introducing the project
- An individual CV webpage
- JavaScript interactive features
- A Bash script that analyses HTML elements
- Consistent styling across webpages using a single CSS file

The project demonstrates beginner to slightly intermediate  front-end development skills and basic GitHub workflow knowledge.

---

 Features

- Navigation bar between pages
- Individual CV webpage
- Current date and time display
- Show / hide CV summary functionality
- Interactive GitHub buttonn
- Bash script for analyzing HTML elements

 Setup Instructions

 1. Clone the Repository

```bash
git clone https://github.com/eaboiniklas-dot/NiklasRassloff_CVportfolio.git
```

 2. Open the Project Folder

```bash
cd NiklasRassloff_CVportfolio
```

 3. Open the Project in Visual Studio Code

```bash
code .
```

 4. Run the Websiite

Open `index.html` in your browser
OR use the Live Server extension in Visual Studio Code.

---

 Running the Bash Script

 Make the Script Executable

```bash
chmod +x Arden_bash.sh
```

 Run the Scriptt

```bash
bash Arden_bash.sh
```

The script analyses `cv.html` and counts:
- Number of div tags
- Number of image tags
- Number of links
- Number of heading tags
